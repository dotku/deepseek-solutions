# DeepSeek Solutions

企业级人工智能解决方案网站，为企业提供安全、稳定、高性能的 AI 应用部署方案。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages。当代码推送到 main 分支时，会自动触发部署流程。

### 手动部署

如果需要手动部署，可以运行：

```bash
npm run deploy
```

## 技术栈

- React
- TypeScript
- Vite
- Tailwind CSS
- GitHub Actions
